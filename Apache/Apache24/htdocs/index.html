PortableWinPy

PortableWinPy index page

PortableWinPy is a portable server for Python web development and testing.

With PortableWinPy you can test deploy-related issues for your WSGI applications on your Windows desktop:

  • Run Apache http server with mod_wsgi interface
  • Turn Debug option in your application on and off
  • Separate serving of static files between Apache and your python code, test static and media paths and urls
  • Play with Apache configuration options
  • Run virtual env python console
  • Run your application tests and stress test in near-production enviroinment - locally

After testing you can wrap and ship your application in single zip file as demo portable package
